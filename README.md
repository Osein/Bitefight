### Installation

Bitefight clone runs under [Laravel](https://laravel.org/) v8.
It need php >= 7.3 to run correctly.

Install the website using those commands :

```sh
$ composer install
$ npm install
$ npm run dev
$ php artisan migrate --seed
$ php artisan serve
```

For production environments...

```sh
$ composer install
$ npm install
$ npm run prod
$ php artisan migrate --seed
```
